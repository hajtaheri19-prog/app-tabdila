# Assets Directory

This directory contains all the assets for the Tabdila app.

## Required Assets

### Fonts
Please add the following Vazir font files to `assets/fonts/`:
- Vazir-Regular.ttf
- Vazir-Bold.ttf
- Vazir-Medium.ttf
- Vazir-Light.ttf

You can download Vazir font from: https://github.com/rastikerdar/vazir-font

### Images
- Logo and app icons should be placed in `assets/images/`
- The main logo is a purple circle icon

### Icons
- Custom icons can be placed in `assets/icons/`

### Animations
- Lottie animations can be placed in `assets/animations/`
